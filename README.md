# MapNUS
Mapping Your Way Through NUS

## Prerqeuisites
1. React (`18.x`)
2. Express (`4.x`)
3. Node.js (`18.x`)
4. npm (`10.x`)
5. neo4j Community Edition (`5.x`)

## Getting Started
1. **Setting up Backend**
```
cd backend
npm install
npm run dev
```

2. Setting up  Frontend
```
cd frontend
npm install
npm run dev
```
